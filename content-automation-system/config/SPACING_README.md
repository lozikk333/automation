# YARA BITES Spacing System

## 🎯 Premium Editorial Layout Rhythm

A comprehensive spacing system designed for luxury recipe websites. Based on an 8px grid system, it creates premium whitespace, elegant visual rhythm, and a calm, balanced composition perfect for editorial food content.

---

## 📐 Spacing Foundation

### Base Unit: 8px

All spacing derives from an 8px grid for precision and premium feel.

| Scale | Value | Units | Usage |
|-------|-------|-------|-------|
| xs | 4px | 0.5 | Micro spacing, tight gaps |
| sm | 8px | 1 | Small gaps, related elements |
| md | 16px | 2 | Standard spacing, default gap |
| lg | 24px | 3 | Medium gaps, breathing room |
| xl | 32px | 4 | Large spacing, premium feel |
| 2xl | 48px | 6 | Extra large, major sections |
| 3xl | 64px | 8 | Large sections, hero areas |
| 4xl | 80px | 10 | Large sections, breathing space |
| 5xl | 96px | 12 | Major sections, editorial gaps |
| 6xl | 120px | 15 | Premium sections, hero spacing |
| 7xl | 160px | 20 | Maximum spacing, largest gaps |

## 📱 Mobile Spacing Rules

### Responsive Scaling Factor
- **Desktop to Mobile**: 0.65x multiplier
- Example: 120px → 72px, 96px → 64px, 48px → 32px

### Mobile Touch Targets
- **Minimum Size**: 44x44px
- **Comfortable Size**: 48x56px
- **Large Button**: 56x64px

### Mobile Padding Rule
Never allow elements to touch screen edges. Minimum 16px padding on mobile.

## 🚀 Quick Start

### 1. Install Tailwind Configuration

```tsx
// tailwind.config.ts
import { tailwindSpacingConfig } from '@/config/tailwind-spacing.config'

export default {
  theme: {
    extend: {
      ...tailwindSpacingConfig,
    },
  },
}
```

### 2. Import CSS Utilities

```tsx
// app/layout.tsx or globals.css
import '@/styles/spacing.css'
```

### 3. Use in Components

```tsx
// Section spacing
<section className="section-major">
  <div className="container-max container-px">
    <h2 className="text-3xl">Title</h2>
    <p className="mt-6">Content</p>
  </div>
</section>

// Card grid
<div className="grid-gap-standard grid recipe-grid">
  {recipes.map(recipe => (
    <article className="card-p-medium">
      {/* Card content */}
    </article>
  ))}
</div>

// Form spacing
<form className="form-spacing">
  <div className="form-group-spacing">
    <label className="form-label-spacing">Email</label>
    <input className="form-input-spacing" />
  </div>
</form>
```

## 🎨 Usage Patterns

### Hero Section Pattern
```tsx
<section className="section-hero py-20 md:py-28 lg:py-32">
  <div className="container-max container-px">
    <div className="hero-max-width mx-auto text-center">
      <h1 className="text-5xl md:text-6xl">Headline</h1>
      <p className="mt-6 md:mt-8 text-lg">Description</p>
      <button className="mt-8 md:mt-10">CTA</button>
    </div>
  </div>
</section>
```

### Recipe Grid Pattern
```tsx
<section className="section-medium">
  <div className="container-max container-px">
    <h2 className="text-4xl mb-8">Recipes</h2>
    <div className="recipe-grid gap-8">
      {recipes.map(recipe => (
        <article className="card-p-medium">
          {/* Content */}
        </article>
      ))}
    </div>
  </div>
</section>
```

### Card with Internal Spacing Pattern
```tsx
<div className="card-p-medium">
  <span className="text-label">Category</span>
  <h3 className="text-heading-lg mt-4">Title</h3>
  <p className="text-body text-secondary mt-3">Description</p>
  <div className="text-meta text-light mt-4">Meta info</div>
  <a href="#" className="link-primary block mt-5">Learn more →</a>
</div>
```

### Two Column Layout Pattern
```tsx
<div className="grid grid-cols-1 lg:grid-cols-2 two-column-spacing">
  <div>{/* Column 1 */}</div>
  <div>{/* Column 2 */}</div>
</div>
```

## ✅ Design Principles

### DO ✅

- ✓ Use the spacing scale consistently
- ✓ Maintain minimum 16px padding on mobile
- ✓ Increase spacing for visual hierarchy
- ✓ Group related elements closely
- ✓ Use whitespace intentionally
- ✓ Scale spacing responsively
- ✓ Keep touch targets ≥ 44px
- ✓ Balance density with breathing room

### DON'T ❌

- ✗ Allow elements to touch screen edges
- ✗ Mix different spacing scales randomly
- ✗ Cramped layouts with tight spacing
- ✗ Oversized gaps that feel chaotic
- ✗ Inconsistent responsive scaling
- ✗ Ignore mobile touch targets
- ✗ Use arbitrary spacing values
- ✗ Break visual rhythm with odd gaps
